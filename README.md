# MiniZiVPN

Modern Android UDP VPN application with Flutter + Android VPN Service.

## Features
- UDP VPN
- SSH Tunnel
- Xray/VLESS
- Live Logs
- AutoPilot
- Traffic Monitor
- Modern Cyber UI

## Stack
- Flutter
- Kotlin
- sing-box
- tun2socks
- GitHub Actions

## Build
```bash
flutter pub get
flutter build apk --release
```
